## How to Populate

**Option A — CSV (manual bank export):**
1. Download a CSV from your bank's web portal
2. Save to the folder in `FINANCE_CSV_DIR` (default: `~/Downloads/bank_statements/`)
3. Run: `python sentinels/finance_watcher.py`

**Option B — Plaid API (automated):**
1. Set `PLAID_ACCESS_TOKEN`, `PLAID_CLIENT_ID`, `PLAID_SECRET` in `.env`
2. Run: `python sentinels/finance_watcher.py --loop`

**CSV format expected:**
```
Date,Description,Amount,Balance
2026-02-23,Grocery store,-45.20,1254.80
2026-02-22,Client payment,+1500.00,1300.00
```
